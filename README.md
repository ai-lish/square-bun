## 📖 玩法

### 🃏 普通版 — 實體卡牌（2–6人）

**準備**
一副 88 張數字卡（1–88），洗勻後枱面發 4 張，其餘做牌庫。

**流程**
1. 玩家輪流擲兩粒骰仔
2. 目標數字 = 可被任一骰仔整除的卡（**不收集**）
3. 其餘玩家觀察枱面，點頭確認後動手收取**非目標卡**
4. 收取後可再揭卡，湊夠 4 張繼續

**平方包**
有平方數（1、4、9、16…）時可搶先拍枱大喊「平方包！」並收取。

**計分**
| 行動 | 得分 |
|------|------|
| 正確收取非目標卡 | +1 分 |
| 成功宣佈平方包 | +2 分 |

最快 **20 分**者勝。

### 🎮 豪華版 — 單機卡牌收集遊戲

**目標**：收集所有 **1–20** 數字卡牌，完成收集後仍可繼續遊玩，挑戰最高**成功率**與**連勝數**。

**準備**
電腦隨機發 **4 張卡**（範圍 1–20，數字不重複）。

**流程**
1. **開卡** — 點擊「🔓 開卡」揭開卡牌，揭示所有數字
2. **擲骰** — 點擊「🎲 擲骰」，畫面顯示「可被 X 或 Y 整除」
3. **揀卡** — 點擊揀選所有**不能被 X 或 Y 整除**的卡；可揀多張，或直接跳過
4. **核對** — 點擊「✓ 核對」確認：
   - ✅ 全部正確 → 卡牌加入收集，**成功+1**、**連勝+1**
   - ❌ 有錯 → **失敗**（成功率 -1）、**連勝歸零**、隨機失去一張已收集卡牌
5. **平方包** — 任意時刻可點擊「🎯 平方包！」按鈾宣佈，**成功+1**；失敗則**連勝歸零**
6. **補卡** — 揭開足夠卡後自動補充至 4 張

**收集系統**
- 📚 **0/20** 按鈕顯示已收集幾多張不同卡牌（點擊可查看收集詳情）
- 已收集的卡可以**重複出現**（牌組範圍隨關卡擴大：1-20 → 1-100 → 1-200 → 1-500 → 1-1000 → 1-2000）
- Penalty 時失去一張已收集卡牌（該卡收集數量 -1）；若某卡全數失去，該卡重新可被收集

**因數圈**
揭開卡牌後，每張卡顯示因數圈：
- 🟢 **綠色** — 該因數喺骰仔中（可被整除）
- 🔴 **紅色** — 該因數唔喺骰仔中（不可被整除）
- 因數圈幫助玩家快速判斷邊張卡係目標（骰仔可整除）或非目標

**成功率與連勝**
- **成功率** = 正確次數 ÷ 總嘗試次數（百分比顯示，實時更新）
- **連勝** = 連續答對/平方包成功次數，答錯或跳過歸零（實時顯示 🔥）
- 兩個數值均**實時顯示**於遊戲畫面頂部

**平方包規則**
1. **任意時刻**都可以宣佈（唔需要先揭開全部卡）
2. 必須枱面**有平方數卡**（1、4、9、16）先可以宣佈
3. 宣佈後結算：
   - ✅ 所有已揭開卡均為平方數 → 收取全部平方數卡，**成功+1**、**連勝+1**
   - ❌ 有卡唔係平方數 → 隨機失去一張已收集卡牌，**連勝歸零**

**📚 收集詳情**
點擊遊戲畫面右上角 **📚 0/20** 按鈕，可查看：
- 已收集卡牌列表（按數字排列）
- 每張卡嘅**收集次數**（同一張卡可多次獲得）
- 不同卡牌數量 / 20 種

**Penalty 機制**
答錯或平方包失敗時，随機失去一張**已收集卡牌**：
- 該卡收集數量 -1
- 若該卡收集次數歸零，該卡**重新可被收集**（有機會再次獲得）

**豪華視覺與音效**
- 🔊/🔇 音效開關（Web Audio API 合成音效，無需外部音頻文件）
- 答對時爆發**金粉粒子效果**
- 答錯時**熒幕震動**
- 平方包成功時**金色閃光**
- 成功率增減時**浮動數字動畫**
