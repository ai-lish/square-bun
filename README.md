## 📖 玩法

### 🃏 普通版 — 實體卡牌（2–6人）

**準備**
一副 88 張數字卡（1–88），洗勻後枱面發 4 張，其餘做牌庫。

**流程**
1. 玩家輪流擲兩粒骰仔
2. 目標數字 = 可被任一骰仔整除的卡（**不收集**）
3. 其餘玩家觀察枱面，點頭確認後動手收取**非目標卡**
4. 收取後可再揭卡，湊夠 4 張繼續

**平方包**
有平方數（1、4、9、16…）時可搶先拍枱大喊「平方包！」並收取。

**計分**
| 行動 | 得分 |
|------|------|
| 正確收取非目標卡 | +1 分 |
| 成功宣佈平方包 | +2 分 |

最快 **20 分**者勝。

### 🎮 豪華版 — 單機卡牌收集遊戲

**目標**：收集非目標卡牌，率先累積 **20 分**取勝。

**準備**
電腦隨機發 4 張卡。

**流程**
1. **擲骰** — 點擊「🎲 擲骰」，畫面顯示「可被 X 或 Y 整除」
2. **揀卡** — 點擊揀選所有**不能被 X 或 Y 整除**的卡；可揀多張，或直接跳過
3. **核對** — 點擊「✓ 核對」確認：
   - ✅ 全部正確 → 收取卡牌，**+1 分**
   - ❌ 有錯 → 退回已揭開的卡，**停止該回合**
4. **平方包** — 桌面有平方數時，點擊「🎯 平方包！」按鈾收取，**+2 分**
5. **補卡** — 揭開足夠卡後自動補充至 4 張

**Combo 系統**
連續答對自動累積 Combo，Combo ≥ 3 顯示 🔥，最高可得 **×2** 分數加成。

**成就系統**
完成特定目標解鎖成就（🔒 → ✅），部分成就需達標 Combo 或使用平方包。

**豪華視覺**
- 音效反饋（正確/錯誤/平方包）
- 粒子效果（答對時爆發金粉）
- 熒幕震動（答錯時）

**勝利條件**
首先累積 **20 分**者勝出。
