## 📖 玩法

### 🃏 普通版 — 實體卡牌（2–6人）

**準備**
一副 88 張數字卡（1–88），洗勻後枱面發 4 張，其餘做牌庫。

**流程**
1. 玩家輪流擲兩粒骰仔
2. 目標數字 = 可被任一骰仔整除的卡（**不收集**）
3. 其餘玩家觀察枱面，點頭確認後動手收取**非目標卡**
4. 收取後可再揭卡，湊夠 4 張繼續

**平方包**
有平方數（1、4、9、16…）時可搶先拍枱大喊「平方包！」並收取。

**計分**
| 行動 | 得分 |
|------|------|
| 正確收取非目標卡 | +1 分 |
| 成功宣佈平方包 | +2 分 |

最快 **20 分**者勝。

### 🎮 豪華版 — 單機卡牌收集遊戲

**目標**：收集非目標卡牌，率先累積 **20 分**取勝。

**準備**
電腦隨機發 **4 張卡**（範圍 1–20，數字不重複）。

**流程**
1. **開卡** — 點擊「🔓 開卡」揭開卡牌，揭示所有數字
2. **擲骰** — 點擊「🎲 擲骰」，畫面顯示「可被 X 或 Y 整除」
3. **揀卡** — 點擊揀選所有**不能被 X 或 Y 整除**的卡；可揀多張，或直接跳過
4. **核對** — 點擊「✓ 核對」確認：
   - ✅ 全部正確 → 卡牌加入收集，**+1 分**（+Combo加成）
   - ❌ 有錯 → 隨機失去一張已收集卡牌，**停止該回合**
5. **平方包** — 桌面有平方數時，點擊「🎯 平方包！」按鈾收取，**+2 分**
6. **補卡** — 揭開足夠卡後自動補充至 4 張

**收集系統**
- 📚 **0/20** 按鈕顯示已收集幾多張不同卡牌（點擊可查看收集詳情）
- 已收集的卡可以**重複出現**（牌組永遠包含所有 1–20）
- Penalty 時失去一張已收集卡牌（該卡收集數量 -1）；若某卡全數失去，該卡重新可被收集

**因數圈**
揭開卡牌後，每張卡顯示因數圈：
- 🟢 **綠色** — 該因數喺骰仔中（可被整除）
- 🔴 **紅色** — 該因數唔喺骰仔中（不可被整除）
- 因數圈幫助玩家快速判斷邊張卡係目標（骰仔可整除）或非目標

**Combo 系統**
- 連續答對累積 Combo，答錯或跳過重置
- Combo ≥ 2：🔥 顯示 Combo 倍率
- Combo ≥ 3：額外 **+1 分**加成
- Combo ≥ 5：**×2** 分數加成（所有分數雙倍）

**成就系統（localStorage 持久化）**

| 🔥 Combo 新手 | 達成 3 連 Combo |
| ⚡ Combo 大師 | 達成 5 連 Combo |
| 🎤 平方初體驗 | 首次使用平方包 |
| 👑 平方獵人 | 使用平方包 5 次 |
| 💯 完璧 | 連續 10 次回答正確 |
| 🧮 因數學者 | 查看因數顯示 50 次 |
| ⏭️ 跳過達人 | 跳過（無夾到）5 次 |
| 🏆 50分高手 | 總得分達到 50 分 |

**分數計算**

| 行動 | 基本分 | Combo 加成 |
|------|--------|------------|

**平方包規則**
1. 必須先**揭開全部桌面卡牌**先可以宣佈
2. 必須枱面**有平方數卡**（1、4、9、16）先可以宣佈
3. 宣佈後結算：
   - ✅ 所有已揭開卡均為平方數 → 收取全部平方數卡，**+2 分/張**
   - ❌ 有卡唔係平方數 → 失去一張已收集卡牌，停止該回合

**📚 收集詳情**
點擊遊戲畫面右上角 **📚 0/20** 按鈕，可查看：
- 已收集卡牌列表（按數字排列）
- 每張卡嘅**收集次數**（同一張卡可多次獲得）
- 不同卡牌數量 / 20 種

**Penalty 機制**
答錯或平方包失敗時，随機失去一張**已收集卡牌**：
- 該卡收集數量 -1
- 若該卡收集次數歸零，該卡**重新可被收集**（有機會再次獲得）
- 若尚未收集過任何卡牌，則直接扣 1 分

**勝利條件與結算**
首先累積 **20 分**者勝出，遊戲結束顯示結算畫面：
- 總得分
- 已收集卡牌數量
- 最高 Combo 等級
- 已解鎖成就列表
