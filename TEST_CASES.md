## A. 遊戲初始化 (5 案例)

### T001 ✅ 正常開始遊戲
**輸入**: 點擊「開始練習」  
**預期**: 桌面生成 4 張卡（預設），階段 = `closed`，所有卡背面朝上  
**驗證**: `phase === 'closed'`，`table.length === 4`，`revealed` 全為 `false`

### T002 ✅ 選擇不同卡牌數量
**輸入**: 切換卡牌數量為 2 → 開始遊戲  
**預期**: 桌面只有 2 張卡  
**驗證**: `table.length === 2`

**輸入**: 切換卡牌數量為 6 → 開始遊戲  
**預期**: 桌面有 6 張卡  
**驗證**: `table.length === 6`

### T004 ✅ 重新開始遊戲重置狀態
**輸入**: 玩一局後，點擊「重新開始」  
**預期**: 分數重置為 0，`wonNumbers` 清空，`table` 重新生成  
**驗證**: `score === 0`，`wonNumbers.size === 0`（或 Map 為空）

### T014 ✅ 兩個骰仔值相同

function getTargetSet(dice) {
  const t = new Set();
  for (const n of ALL_NUMBERS) {
    if (dice.some(v => v !== null && n % v === 0)) t.add(n);
  }
  return t;
}
