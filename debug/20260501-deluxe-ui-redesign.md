# Square Bun 豪華版 UI 重新設計方案

**日期：** 2026-04-29

## 📋 概覽

兩個主要改動：
1. **卡牌收藏** → 新增 📚 收藏按鈕，點擊彈出 modal 網格展示
2. **成就顯示** → 移除右下角 badge bar，改為 toast notification

---

## 🃏 卡牌收藏 UI

### 入口
- Header 右側新增 📚 按鈕
- 顯示已收集數量：`📚 12/20`
- Hover: 輕微放大動畫

### Modal 設計
- 全屏半透明黑色遮罩
- 中央白色/深色卡牌框
- 標題：`📚 卡牌收藏 (12/20)`
- 關閉按鈕（右上角 X 或點擊遮罩關閉）

### 卡牌網格
- 響應式網格：桌面 5-6 列，平板 4 列，手機 3 列
- 每張卡顯示：
  - 卡牌數字（大字體居中）
  - 右下角 quantity badge（紅色圓形，如 `×3`）
  - 未收集的卡：顯示灰色鎖頭 + 問號
- 卡牌尺寸：60px × 80px（迷你版，無需顯示因素圈）

### 卡牌狀態
| 狀態 | 顯示 |
|------|------|
| 已收集 | 正常彩色卡 + 數量 badge |
| 未收集 | 灰色半透明 + 🔒 + `?` |

### 互動
- 點擊已收集卡牌：放大顯示（可見因素圈 + 收集次數）
- 拖拽/滾動瀏覽大量卡牌
- 右上角 `X` 或點擊遮罩關閉

---

## 🏆 成就顯示 UI

### Toast Notification（右上角）

**尺寸**：280px × 80px
**位置**：右上角，距邊 20px
**動畫**：由右滑入 → 停留 3 秒 → 滑出右側

**設計**：
```
┌─────────────────────────────┐
│ 🏆 成就解鎖！                │
│ 「Combo 新手」3連勝！        │
└─────────────────────────────┘
```

**狀態**：
- 背景：深色半透明（`rgba(20,20,30,0.95)`）
- 邊框：金色（成就專用色）
- 字體：白色標題 + 金色成就名

**堆疊**：多個 toast 時垂直堆疊，間隔 10px

### 成就列表（獨立頁面）

**入口**：Header 左側 🏆 按鈕，顯示已解鎖/總數（如 `3/6`）

**Modal 設計**：

- 標題：`🏆 成就 (3/6)`
- 成就列表（可滾動）：

## 🎬 動畫設計

### Toast 動畫（CSS）
```css
@keyframes slideInRight {
  from { transform: translateX(120%); opacity: 0; }
  to   { transform: translateX(0);    opacity: 1; }
}
@keyframes slideOutRight {
  from { transform: translateX(0);    opacity: 1; }
  to   { transform: translateX(120%); opacity: 0; }
}
.toast { animation: slideInRight 0.4s ease-out; }
.toast.hiding { animation: slideOutRight 0.3s ease-in forwards; }
```

### Modal 動畫
```css
@keyframes fadeIn {
  from { opacity: 0; }
  to   { opacity: 1; }
}
.modal-overlay { animation: fadeIn 0.2s ease-out; }
```
